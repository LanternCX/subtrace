# Subroute 实现计划

## 目标

完成一个可在 macOS 上运行的 `subroute` 命令行工具，能够从 Base64 订阅地址生成节点入口去程路线 Markdown 报告。

## 任务拆分

### 任务 1：初始化项目骨架

- 建立 Rust 命令行项目。
- 补齐最小依赖与模块边界。
- 明确输入、输出和主流程入口。

### 任务 2：实现订阅获取与解析

- 拉取远程订阅内容。
- 完成 Base64 解码。
- 逐行解析节点条目。
- 产出可测节点列表和无法解析列表。

### 任务 3：实现基础探测能力

- 调用系统 `ping` 获取基础连通性信息。
- 调用系统 `traceroute` 获取逐跳原始结果。
- 解析逐跳原始结果中的跳数、延迟与有效 IP。

### 任务 4：实现逐跳标注

- 汇总所有有效跳点 IP。
- 对公开 IP 查询 AS 与归属地。
- 对私网、本地地址和查询失败情况给出清晰标注。

### 任务 5：实现报告输出

- 生成面向人工阅读的 Markdown 报告。
- 报告中同时展示顶部摘要、节点明细和异常情况。
- 输出默认报告文件路径。

### 任务 6：补齐验证

- 为订阅解析、逐跳解析和报告渲染补充自动化测试。
- 在本机运行测试、格式化和完整构建。
- 修正验证阶段发现的问题。
