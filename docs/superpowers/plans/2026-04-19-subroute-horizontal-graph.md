# Subroute 横向关系图实现计划

## 目标

将当前 HTML 页面中的纵向关系图改成横向关系图，并把末端机场节点按最终 AS 聚合为可展开的组节点。

## 任务拆分

### 任务 1：重建图模型

- 保留全局唯一的 AS 节点。
- 图结构按关系图表达，不再伪装成树。
- 末端机场节点改为按最终 AS 聚合的组节点。

### 任务 2：重排图布局

- 将图从纵向布局改为横向布局。
- 保持缩放和点击交互。
- 允许共享节点和交叉连线。

### 任务 3：接入末端分组展开

- 点击组节点后展示组内机场节点列表。
- 点击具体机场节点后仍展示该节点的路由明细。
- 保持底部详情区与图交互联动。

### 任务 4：补齐验证

- 为横向布局、唯一 AS 节点和组节点详情补充自动化测试。
- 运行测试、格式化和构建。
- 用真实订阅生成新的 HTML 页面验证输出结构。
