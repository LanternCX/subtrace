# Subroute HTML 关系图实现计划

## 目标

将当前 Markdown/SVG 结果替换为单一 HTML 页面，使用唯一 AS 节点的关系图作为主视图，并在页面内直接查看节点详细信息。

## 任务拆分

### 任务 1：重建唯一 AS 图模型

- 改为同一个 AS 在全图中只保留一个节点。
- 保留机场节点作为终点节点。
- 允许真实关系带来的交叉和回连。

### 任务 2：生成 HTML 关系图页面

- 输出单一 HTML 文件作为主结果。
- 页面整体按从上到下布局。
- 中间展示关系图，底部展示详情。

### 任务 3：实现交互与筛选

- 点击 AS 或机场节点后展示对应详情。
- 提供按地区粗分组的筛选和对象列表。
- 支持搜索、高亮和缩放浏览。

### 任务 4：保留节点明细

- 在详情区继续展示机场节点的路由明细表。
- 字段继续保持 `IP / 地区 / AS 号 / 组织`。

### 任务 5：补齐验证

- 为唯一 AS 图、HTML 引用和详情展示补充自动化测试。
- 运行测试、格式化和构建。
- 用真实订阅生成一次 HTML 页面并检查主要内容。
